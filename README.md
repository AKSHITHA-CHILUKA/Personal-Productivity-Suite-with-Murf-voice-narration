## How to Use

1. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
2. Set your Murf API key in each app file.
3. Run the dashboard:
    ```
    streamlit run app.py
    ```
4. Use the sidebar to navigate between apps.
